# KULTIVAIT Evaluation Framework

## Goals
- Evaluate performance of different AI models for various tasks.
- Optimize configurations for best results.

## Criteria
1. **Accuracy**: Measure correctness of outputs.
2. **Efficiency**: Time taken to complete a task.
3. **Cost**: Computational resources used (CPU, memory) and cost if applicable.
4. **Usability**: Ease of integration with existing workflows.

## Experimental Setup
### Task Types
1. Research-based tasks: Literature review, summarization.
2. Documentation-based tasks: Writing technical documentation, creating tutorials.
3. Coding tasks: Code generation, debugging.
4. CI Management tasks: Reviewing PRs, running tests.
5. Deployment operations: Managing environment configurations, deploying changes.

### Models to Test
- agy gemini flash 3.5
- claude opus/fable
- qwen 3,4/5 9B

### Benchmarks and Metrics
1. Define benchmarks for each task type.
2. Set up metrics (e.g., accuracy score, completion time) to measure performance.

### Data Collection Methods
1. Collect data from experiments using specified models.
2. Document findings and results.
3. Analyze the collected data against set criteria.
4. Iterate based on findings to optimize configurations for best performance.
