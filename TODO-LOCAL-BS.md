# Evaluating and Orchestrating AI Agents

## Goal:

Come up with a manner in which we can experiment running a variety of research-based, documentation-based, coding tasks, repository pull-request CI management, and deployment operations that will give us the datapoints needed to prove our worth.

## Tasks:

### Research-Based Tasks
- Define specific research questions.
- Evaluate accuracy and efficiency using agy gemini flash 3.5, claude opus/fable, and qwen 9B.

### Documentation-Based Tasks
- Write technical documentation and tutorials.
- Measure usability in integrating with existing workflows.

### Coding Tasks
- Code generation and debugging tasks.
- Compare efficiency between different models (agy gemini flash 3.5 vs claude opus/fable).

### CI Management Tasks
- Reviewing PRs, running tests.
- Evaluate cost and computational resources used.

### Deployment Operations
- Manage environment configurations.
- Deploy changes and measure time taken.

## Evaluation Criteria:

1. **Accuracy** - Measure correctness of outputs.
2. **Efficiency** - Time taken to complete a task.
3. **Cost** - Computational resources used (CPU, memory) and cost if applicable.
4. **Usability** - Ease of integration with existing workflows.

## Experimental Setup:
- Define benchmarks for each task type.
- Specify how different models will be tested on each benchmark.
- Plan data collection methods and analysis steps.

## Next Steps:
1. Implement research-based experiments.
2. Evaluate documentation tasks using specified criteria.
3. Conduct coding challenges to compare efficiency across models.
4. Setup CI management tests for PR reviews and test runs.
5. Deploy changes in different environments and measure deployment times.
